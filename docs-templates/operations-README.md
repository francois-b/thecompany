# Operations

Runbooks, deployment guides, and operational procedures.

## What Goes Here

- Deployment procedures
- Incident response playbooks
- Infrastructure setup guides
- Monitoring and alerting configuration
- On-call procedures

## Template

```markdown
# Runbook: Title

**Last Updated:** YYYY-MM-DD
**Owner:** Name or team

## Overview

What this runbook covers and when to use it.

## Prerequisites

- Required access
- Required tools
- Required knowledge

## Procedure

### Step 1: Description

```bash
# Commands to run
```

Expected output or result.

### Step 2: Description

Details and commands.

## Rollback

How to undo if something goes wrong.

## Troubleshooting

### Common Issue 1

Symptoms and resolution.

### Common Issue 2

Symptoms and resolution.

## Related

- Links to related runbooks or docs
```

## Naming Convention

Use descriptive names: `deployment.md`, `incident-response.md`, `database-backup.md`
