# Design Docs

Design documents for proposals, decisions, and implementation records.

## Creating a New Design Doc

Use the slash command:

```
/new-design-doc
```

Or manually:

1. Find the next number: `~/.claude/scripts/next-doc-id.sh DES`
2. Create file: `DES-NNN-kebab-title.md`
3. Copy template from [project-documentation-structure.md](../meta/project-documentation-structure.md)

## Template

See the [Project Documentation Structure Guide](../meta/project-documentation-structure.md#design-doc-template) for the full template.

## Status Legend

| Status | Meaning |
|--------|---------|
| **Draft** | Work in progress |
| **Reviewing** | Open for feedback |
| **Accepted** | Ready to implement |
| **Implemented** | Complete |
| **Deprecated** | Superseded |
