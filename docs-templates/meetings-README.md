# Meetings

Meeting notes, standups, and discussion records.

## Naming Convention

Use date-prefixed names: `YYYY-MM-DD-description.md`

Examples:
- `2025-12-08-standup.md`
- `2025-12-08-sprint-planning.md`
- `2025-12-08-design-review.md`

## Template

```markdown
# Meeting: Description

**Date:** YYYY-MM-DD
**Attendees:** Names or "solo"

## Agenda

1. Topic one
2. Topic two

## Notes

### Topic One

Discussion points and decisions.

### Topic Two

Discussion points and decisions.

## Action Items

- [ ] Action item with owner
- [ ] Another action item

## Follow-up

Next meeting date or async follow-up plan.
```

## Quick Standup

For daily standups, use the `/standup` command which auto-generates a project status overview.
